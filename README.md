# Introducción
Este proyecto se basa en analizar a través de preguntas complicadas la base de datos Sakila. A través de distintos filtrados, subconsultas y uniones de tablas queremos que el cliente sea capaz de buscar todo lo que necesite en la base de datos.
Para ello hemos utilizado lenguaje SQL.

# Estructura de las consultas
El cliente comenzó con unas consultas básicas sobre las tablas film y actor. Que fueron volviendose más complejas conforme iba avanzando. Utilizando WHERE y LIKE para hacer las consultas. Más adelante utilizamos agregaciones con GROUP BY para hacer recuentos, promedios o por ejemplo conocer los clientes con mayor volumen de alquileres. Para terminar usamos consultas avanzadas como JOIN o subconsultas cruzando muchos datos a la vez.

# Conclusión
Gracias a este proyecto hemos identificado el número total de películas por clasifiicación de edad o los clientes con más alquileres de películas y más fieles al videoclub.
